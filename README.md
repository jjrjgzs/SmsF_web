# 🚀 快速开始

## 1️⃣ 本地配置（手机端）
1. **下载文件到**  
`Download`
2. **打开 SmsForwarder**  
`服务端->Web客户端->选择目录->选择刚下载的文件`

## 2️⃣ 服务器部署
- 将文件上传至您的服务器即可使用。

## 3️⃣ 生成自定义 APP
